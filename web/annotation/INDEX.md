# Annotation Examples (Local)

These are runnable examples you can clone and run locally. Each demonstrates specific annotation features in Nutrient Web SDK.

> For interactive playground examples, see [Annotations Playground Examples](../../docs/examples/web-annotations.md).

---

## Available Examples

### comment-annotations
**Path**: `./comment-annotations/`
**Tech Stack**: Plain JavaScript
**Description**: Demonstrates how to create and manage comment annotations on PDFs.

**Quick start**:
```bash
cd comment-annotations
npm install
npm start
```

---

## Prerequisites

All examples require:
- **Node.js** (v16 or higher recommended)
- **npm** or **yarn**
- A valid **Nutrient license key** (see individual READMEs for configuration)
